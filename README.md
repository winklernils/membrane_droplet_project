# Membrane Droplet Project

## Overview

This project investigates membrane-associated droplet dynamics using computational modeling.
It is part of my PhD research in theoretical/computational physics.

The focus is on:

* phase separation and droplet formation
* membrane interactions
* numerical simulation of underlying physical models

---

## Project Structure

* `src/` – core simulation code and models
* `notebooks/` – exploratory analysis and experiments
* `config/` – configuration files (parameters, setups)
* `data/` – input data (ignored by Git)
* `results/` – simulation outputs (ignored by Git)
* `figures/` – generated plots (ignored by Git)

---

## Setup

Create a Python environment and install dependencies:

```bash
pip install -r requirements.txt
```

*(Environment setup will be refined later.)*

---

## Usage

Run a simulation (example):

```bash
python src/main.py
```

---

## Reproducibility

* All simulation parameters should be stored in `config/`
* Results and figures are **not tracked** and must be reproducible from code
